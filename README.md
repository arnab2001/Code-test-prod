## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/chhakuli123/code-compass.git
```

2. Install dependencies
```bash
$ cd code-compass
$ npm install # Or yarn install
```

3. Start the Frontend:
```bash
 $ npm run dev # Or yarn dev
```
